# To-Do List Project 📝

¡Bienvenida a mi primer proyecto de Lista de Tareas! Este es un desarrollo realizado con HTML, CSS y JavaScript puro (Vanilla JS), enfocado en la manipulación dinámica del DOM.

## 🚀 Demo en vivo
Puedes ver el proyecto funcionando aquí: [lista-de-tareas-zeta-wine.vercel.app](https://lista-de-tareas-zeta-wine.vercel.app)

## ✨ Características
* **Agregar tareas:** Captura el texto del input y lo añade a una lista visual.
* **Eliminar tareas:** Cada tarea cuenta con un botón "X" que permite removerla de forma individual.
* **Diseño Responsivo:** Adaptado para verse bien tanto en computadoras como en dispositivos móviles.

## 🛠️ Tecnologías utilizadas
* **HTML5:** Estructura semántica.
* **CSS3:** Estilos personalizados, gradientes y diseño flexible (Flexbox).
* **JavaScript:** Lógica funcional, creación dinámica de elementos (`createElement`) y manejo de jerarquías de nodos (`appendChild`).

## 🧠 Aprendizajes clave
Durante este proyecto, puse en práctica conceptos fundamentales:
1.  **Jerarquía del DOM:** Entender la relación Padre/Hijo para organizar los elementos (la lista como padre de las tareas, y las tareas como padres del botón eliminar).
2.  **Eventos:** Uso de `onclick` para disparar funciones de agregar y remover elementos.
3.  **Git & GitHub:** Manejo de repositorios, solución de conflictos en la terminal y despliegue continuo con Vercel.
